# bootdev-ai-agent-python
A guided project from Boot.dev's Python back-end curriculum, building an AI agent from scratch.

This project was built as part of the [Build an AI Agent in Python](https://www.boot.dev/courses/build-ai-agent-python) guided project on [Boot.dev](https://www.boot.dev). It walks through constructing an AI agent from scratch in Python, covering the core concepts of how agents reason, use tools, and act on instructions. The project is a practical complement to the Boot.dev back-end development curriculum.
